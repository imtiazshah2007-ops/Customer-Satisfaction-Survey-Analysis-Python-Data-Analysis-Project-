Survey Satisfaction Analysis

This project analyzes customer satisfaction survey data using Python.
It loads responses from a CSV file, calculates satisfaction statistics, and visualizes results using charts.

The goal of this project is to demonstrate basic data analysis, visualization, and data processing skills using Python.
**Features**
Load survey data from a CSV file
Analyze satisfaction levels
Display counts of each satisfaction category
Generate bar chart visualization
Generate pie chart visualization
Automatically save charts as image files
**Technologies Used**
_Python
Pandas
Matplotlib
Seaborn_
